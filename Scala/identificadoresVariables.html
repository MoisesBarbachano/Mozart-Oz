Variables

Las variables son como los valores, excepto que puedes reasignalas. Puedes definir una variable con la palabra clave "var"

var x = 1 + 1

x = 3

println(x*x) //9

Como con los valores, tú puedes indicar explicitamente el tipo:

var x: Int = 1 + 1
