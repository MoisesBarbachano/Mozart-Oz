Representación Semántica

Compuesta por expresiones, intrucciones de control y reglas de ejecución

Expresiones

1.Operadores

Scala permite operadores como:

Operadores Ejemplos
Aritméticos +,-,*,%,/
Relacionales >,<,>=,=<,!=,==
Lógicos &&,||,!

2.Tipos de datos

En Scala los tipos de datos son clases

Tipos Ejemplos
Byte 8 bit con signo
Short 16 bit con signo
Int 32 bit con signo
Char 16 bit con signo
Float 32 bit con signo
Double 64 bit con signo
Long 64 bit con signo

3.Evaluación de expresiones

Dentro de una expresión, los operadores se evalúan de acuerdo a su orden de preferencia. De mayor preferencia a menor

Evaluación
* / %
+ :<
<>
&
